Yo! Thanks for using Strapforge to get Musdash started!

What is this?

Strapforge is a Yeoman generator built by Alex Ward, it is a basic static site scaffold with a very light SASS framework built into it.

How do I use it?

It's simple! If you are building a completely static site, then just delete/modify the content in this file and the content in the /src/ folder, delete the /public/app folder and you are off! All the dependencies to get a basic site up and running are right here and were installed by Yeoman. So you can get going right away!

Alternatively if you are building a site with a router (e.g. you are using an MVC layout, then delete this file and navigate to /public/app and you will find a file in there with more instructions.

A few notes:

  • I don't believe in BEM notation, so you won't find it in this project. My reasons are simply that I believe it creates uncessesary mark-up when you can achive the same result with a more orderly file my simply writing neat, semantic HTML.
  • I haven't used a framework like Bootstrap or Foundation, this is for the exact same reason as above, the extra markup and bloat created should not be on a production site.
  • I haven't used an off-the-shelf gridsystem. This is because I've written my own flexible grid-system using FlexBox. It even has fallbacks in place so that it should work nicely in browsers as early as IE8!
